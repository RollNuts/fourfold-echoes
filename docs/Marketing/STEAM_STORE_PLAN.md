# Steam Store Plan

Status: D-020 planning document.

This document is the Steam-facing copy and trailer plan for the current D-020
product direction: Steam first, buy-to-play, single-player, compact top-down
action-adventure, one hub, up to three regions, up to four bosses, and one
exploration tool.

## 短文紹介3案

1. **Primary candidate, 152 chars:** Explore a compact top-down fantasy
   adventure where one mysterious tool opens paths, reveals hidden routes, and
   creates boss openings.
2. **Alternate, 138 chars:** Leave a small hub, cross three handcrafted regions,
   solve tool-based rooms, open shortcuts, and defeat four readable bosses.
3. **Alternate, 119 chars:** A focused single-player action-adventure with one
   hub, one exploration tool, compact regions, and clear boss fights.

## 長文紹介

FOURFOLD ECHOES is a compact single-player top-down action-adventure for Steam.

From a small hub, enter up to three handcrafted fantasy regions, learn one
mysterious exploration tool, open shortcuts, find relic rewards, and defeat four
readable bosses. The adventure is built around a small rule set used repeatedly:
move, dodge, strike, read enemy tells, and use the same tool to solve rooms,
reveal routes, and create brief boss openings.

The first seconds of the store trailer and capsule video must show the actual
game: top-down movement, the hero, an enemy, a tool target, and a route or boss
opening changing on screen. Store copy should promise only what is implemented
or locked by D-020. When capture quality is not ready, delay the claim instead
of filling the page with future-facing wording.

## トレーラー絵コンテ 45秒版

The first 10 seconds must explain the game without narration: top-down hero,
small hub, tool use, route change, and an enemy or boss threat.

| Time | Beat | Capture Requirement | On-Screen Text |
| --- | --- | --- | --- |
| 0-3s | Hero moves through the hub toward region gates | Actual gameplay camera; title not full-screen yet | "A compact top-down adventure" |
| 3-7s | Tool pulse activates a nearby node and opens a path | The same frame shows player, node, effect, and route response | "One mysterious exploration tool" |
| 7-10s | Enemy tell, dodge, and counterattack | Readable threat before the hit | "Read the danger. Strike back." |
| 10-17s | First room puzzle: blocked route becomes passable | No menu explanation; visual cause and effect only | "Open paths and shortcuts" |
| 17-24s | Second room uses the same tool differently | Prove depth without implying extra tools | "Use the same tool in new situations" |
| 24-31s | Relic reward, then immediate play impact or return route | No inventory screen | "Find relic rewards" |
| 31-40s | Boss tell, dodge, tool opening, punish | Production-intent boss silhouette and VFX only | "Face readable bosses" |
| 40-45s | Title, Steam wishlist, demo status if available | Honest date/platform text only | "Wishlist on Steam" |

Audio plan:

- Use one edited gameplay music bed.
- Keep tool pulse, hit confirm, reward, shortcut, and boss tell SFX audible.
- Do not use narration unless it clarifies timing after gameplay already reads.
