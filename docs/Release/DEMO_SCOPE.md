# Demo Scope

Status: D-020 draft.

## Demo Goal

Prove in 20-30 minutes that FOURFOLD ECHOES is a compact top-down
action-adventure whose fun comes from one exploration tool, readable combat,
shortcuts, relic rewards, art, and audio.

## Included

- hub 1
- exploration area 1
- normal enemies 2
- miniboss 1
- boss 1
- exploration tool 1
- shortcut 1
- gimmick rooms 2
- relic rewards 2
- minimal UI
- BGM 2 tracks
- minimum SFX set
- local save/load
- controller support
- store-quality screenshots from actual gameplay

## Excluded

- open world
- co-op
- online account
- multiple exploration tools
- inventory
- crafting
- quest log
- social systems
- multiple regions in demo
- endgame systems
- PS5 build

## Playthrough Checklist

| Beat | Pass Condition |
| --- | --- |
| Hub | player understands return point and region entrance |
| First movement | traversal and camera feel readable |
| First combat | attack, dodge, and hit confirm are satisfying |
| Tool introduction | player sees tool pulse and target response |
| Gimmick room 1 | one tool opens/reveals a route |
| Gimmick room 2 | same tool is used differently |
| Shortcut | opened state is visible and saved |
| Miniboss | readable escalation before boss |
| Boss | clear tells, retry, and tool-related opening |
| Reward | two relic rewards feel meaningful without inventory UI |

## Demo Kill Criteria

The demo is not ready if:

- the hero, tool, enemy, boss, or room still look like gray-box placeholders
- music or SFX are obvious temp tones
- camera hides combat
- the tool is visually unclear
- the same tool does not create at least two distinct room moments
- screenshots need explanatory text to make sense
