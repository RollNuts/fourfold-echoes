## Prepare Now

| Area | Requirement |
| --- | --- |
| Input | controller-first, remappable, no keyboard-only gameplay |
| Performance | explicit CPU/GPU/texture/mesh budgets |
| Memory | avoid unbounded asset loading and unmanaged caches |
| Save | versioned local save, suspend/resume-safe design |
| Platform | abstract achievements, cloud, rich presence, controller glyphs |
| HDR/Display | define target modes later; avoid baked unreadable contrast |
| Build | deterministic validation and asset reports |

## Reopen PS5 Work When

- Steam demo is strong.
- Windows build is stable.
- Store screenshots/trailer prove demand.
- Platform access is available.
