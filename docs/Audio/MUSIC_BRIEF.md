# Music Brief

Status: D-020 draft.

## Role

Music must make a compact top-down adventure feel memorable without hiding
combat or tool feedback.

## Required Cues

| Cue | Role | Status |
| --- | --- | --- |
| `music.hub_or_region01` | hub/first exploration loop | needed |
| `music.first_boss` | first boss fight | needed |
| `stinger.discovery` | secret or shortcut reveal | prototype only |
| `stinger.reward` | relic reward pickup | prototype only |

## Direction

- one short memorable motif
- warm hub version
- more driven exploration version if budget allows
- boss loop with clear downbeats for attack tells
- discovery/reward stingers short enough not to interrupt control
- no complex stem system for MVP

## Acceptance

The vertical slice is not complete until hub/Region 01 music, first boss music,
tool SFX, combat SFX, reward stinger, and UI basics are audible and mixed.
