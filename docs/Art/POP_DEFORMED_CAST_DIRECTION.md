# Pop Deformed Cast Direction

Status: active character/encounter concept direction.

## Decision

The cast should be more pop/deformed and easier to model than realistic fantasy
characters.

Avoid generic class language. The game should not look like a starter pack of
sword boy, green archer, red witch, blonde cleric, and knight tank. The player
roles can keep those gameplay functions, but the visual logic must come from
this game's world.

## World Logic For Cast

The current field direction is block/chunk assembly. Characters and enemies
should therefore borrow from:

- measuring tools
- snap-lines
- folding plates
- stamps and seals
- repair clamps
- portable sockets
- backpacks and trays
- map rigs
- block lifting and route marking
- visible weak points and pressure seams

This gives the cast a local production logic instead of average fantasy class
costumes.

## Proportions

- 2.8 to 3.2 heads tall
- large heads and hands
- compact torsos
- chunky boots
- big readable role props
- simple side-view massing
- broad color blocks, not fine filigree

## Playable Role Translation

Use gameplay roles, not genre costume defaults:

- Lead: route-fixer / snapline toolbearer, not sword boy
- Heavy: block press / guard clamp carrier, not knight tank
- Caster: seal-stamp apparatus user, not robe-and-staff mage
- Scout: traversal spool / range marker user, not green hood archer

## Enemy Translation

Enemies should read by encounter job:

- fodder melee: small forward-lunge body
- shield blocker: obvious protected front
- ranged hazard: clear projectile barrel or seed socket
- charger: low body, strong forward mass, pre-charge tell
- support/healer: obvious buff core or repair arm
- elite: same family, larger tell surfaces and stronger contrast

Avoid:

- goblin/slime/dragon/skull defaults
- animal copied literally from nature
- generic armored knight armies
- mascot face dependency

## NPC Translation

NPCs should show service role through local tools:

- merchant: tray/stall rig and inventory bundles
- blacksmith/upgrade: clamp forge and visible upgrade blocks
- cartographer/quest guide: fold-out map rig and route markers
- healer: patch kit / socket purifier
- trainer: practice block and target rig

Avoid generic fantasy village costumes unless the role prop makes the service
clear from gameplay camera distance.

## Keep-All Rule

Generated sheets are kept even when full designs fail. Mine them for hair,
hats, backpacks, gloves, boots, color blocking, silhouettes, enemy attack reads,
weak points, and service props.
