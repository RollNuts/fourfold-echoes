# D021 Out Of Scope

Status: required rejection list.

## Scope-Out List

| Feature | Decision | Reason |
| --- | --- | --- |
| Open world | Out | The product is compact and authored. |
| Seamless world streaming | Out | It delays the slice and weakens handcrafted room mastery. |
| Procedural world generation | Out | Not needed before authored rooms prove fun. |
| Multiplayer/co-op/PvP | Out | Too much design, sync, and QA cost for v1. |
| Online accounts/backend | Out | Conflicts with offline buy-to-play target. |
| Live service | Out | Not a v1 business model. |
| Inventory | Out | Creates UI/economy scope that the game does not need. |
| Crafting | Out | Pulls attention from rooms, tool mastery, and bosses. |
| Quest log | Out | Implies content scale beyond the MVP. |
| Equipment economy | Out | Not required for the one-tool adventure loop. |
| Hack-and-slash loot rarity/builds | Out | Historical direction, not D021. |
| Extraction loss loop | Out | Historical direction, not D021 player promise. |
| Day/night cycle | Out | High QA cost with weak core value. |
| Fishing/farming/base building | Out | Different genre loops. |
| Second exploration tool | Out | Dilutes mastery of the one core tool. |
| Extra hub | Out | Cap is one hub. |
| Fourth region | Out | Cap is three regions. |
| Fifth boss | Out | Cap is four bosses. |
| Cosmetic shop/gacha | Out | Conflicts with buy-to-play and scope. |

## Allowed Improvements

Allowed improvements must improve an existing D021 item:

- More readable enemy tell.
- Better boss opening clarity.
- Better tool target readability.
- Better controller flow.
- Better pause/settings/save UX.
- Better art, animation, lighting, VFX, or audio for an approved room.
- Better shortcut reveal or reward beat.
- Better Steam screenshot/trailer truthfulness.

## Rejection Rule

Reject a proposal when any of the following are true:

- It exceeds a D021 cap.
- It adds a new screen category.
- It requires another persistent database.
- It creates a player-facing promise that cannot be shown in the current build.
- It makes the first 15 minutes harder to understand.
- It delays the next playable build without raising trailer value.
