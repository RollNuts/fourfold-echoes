# D021 Steam Store Plan

Status: public-copy candidate. All copy must match implemented or locked scope.

## Short Copy Ideas

1. Explore a compact top-down fantasy adventure where one mysterious tool opens
   paths, reveals secrets, and creates boss openings.
2. Leave a small hub, cross handcrafted regions, solve tool-based rooms, open
   shortcuts, and defeat readable bosses.
3. A focused single-player action-adventure built around one tool, compact
   rooms, clear combat, and boss mastery.

## Long Copy

FOURFOLD ECHOES is a compact single-player top-down action-adventure. Start from
a small hub, enter handcrafted regions, use one exploration tool to reveal paths
and open shortcuts, then face bosses built around readable tells and clear
punish windows.

The game is designed for a focused buy-to-play release. No live-service
requirements, no online account, no crafting grind, and no sprawling open-world
promise. The value comes from a small rule set used well: movement, attack,
dodge, one tool, smart rooms, satisfying shortcuts, and boss mastery.
