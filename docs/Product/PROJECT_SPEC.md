# Fourfold Echoes Project Spec

Status: canonical after D-020.

## Executive Summary

FOURFOLD ECHOES is a Steam-first, buy-to-own, single-player top-down classic
action-adventure.

The product is compact by design. It is not an open world and not a systems
sandbox. The core experience is:

> leave a small hub, enter one of three handcrafted regions, master one
> exploration tool, fight readable enemies and bosses, open shortcuts, earn a
> small number of meaningful relic rewards, and return stronger in player skill
> rather than system count.

## Final Product Thesis

Make a compact fantasy adventure that can be understood in a 30-second silent
clip: a readable hero, a visible exploration tool, clear rooms, obvious enemies,
simple combat, and one repeated exploration rule that keeps producing new
room-level surprises.

The hook is not a list of features. The hook is:

- one iconic exploration tool
- room and shortcut design built around it
- bosses that test the same readable rules
- stylized art and audio strong enough to sell screenshots

## MVP Limits

| Scope | Limit |
| --- | --- |
| Hub | 1 |
| Regions | 3 |
| Bosses | 4 total, including final |
| Exploration tools | 1 |
| Vertical slice length | 20-30 minutes |
| Full v1 target | compact premium indie, not content treadmill |
| Multiplayer | none |
| Online services | none |

## Target Audience

- Steam players who want a compact handcrafted adventure.
- Players who prefer readable positioning and room mastery over complex combo
  execution.
- Players who respond to stylized fantasy and clear screenshots.
- Players who want a complete buy-to-play game, not a service.

## Platform Order

1. Steam Windows.
2. Steam Deck compatibility target.
3. Later console feasibility, only after the Steam version is strong.

## Core Controls

- Move
- Aim/facing
- Normal attack
- Dodge
- Guard or damage-reduction action if it proves necessary
- One exploration tool
- Interact
- Pause

Do not add extra action verbs unless one existing verb is removed.

## Progression

Progression is flag-based and compact:

- boss defeated
- region unlocked
- shortcut opened
- relic reward acquired
- save point reached

No inventory, crafting, quest log, social system, gear spreadsheet, or random
loot economy is part of MVP.

## Art Target

Stylized 3D, top-down readable, low-poly-leaning but polished. Shapes, color,
lighting, animation, and VFX must communicate gameplay without explanatory UI.

## Audio Target

Audio is part of readability:

- tool use and response
- hit confirm
- enemy tells
- boss transitions
- shortcut opening
- reward/discovery
- UI confidence

Final validation cannot use placeholder SFX or silent market captures.

## Non-Goals

- Open world.
- Echo Phase/world-state switching.
- Inventory, crafting, quest log, social systems.
- Multiplayer, co-op, PvP, MMO, live service.
- Day/night cycle.
- Fishing, farming, base building.
- Procedural world generation as a product pillar.
- More than 3 regions, 4 bosses, or 1 exploration tool in MVP.
